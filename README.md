# PILGRIM - Comptoir Végétal & Concept Store

## Présentation
PILGRIM est un lieu hybride situé à Pézenas, mêlant cuisine vivante et boutique d'objets choisis avec soin. Le projet s'inscrit dans une philosophie "Slow Life", privilégiant la qualité, le circuit court et le bien-être.

- **Nom** : PILGRIM
- **Concept** : Comptoir Végétal & Concept Store
- **Slogan** : Slow Life, Slow Food, Vegetarian Food
- **Philosophie** : 100% Végétal, Circuit Court, Qualité, Bienveillance
